# 🌐 UI GLOBAL – Règles communes

## Layout principal

- Sidebar gauche (navigation)
- Header top (profil, notifications)
- Content area

## Composants standards

### Table
- Pagination obligatoire
- max 50 lignes

### Modal
- utilisé pour actions rapides

### Form
- validation instantanée
- messages en français

## UX règles

- 3 clics max pour action critique
- boutons visibles
- gros champs (usage terrain)

## États

- loading
- empty
- error

## Sécurité UI

- cacher les éléments non autorisés
- jamais exposer données d’un autre rôle
